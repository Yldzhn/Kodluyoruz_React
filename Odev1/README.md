># Görev Listem
## Amaç
Kullanıcıdan almış olduğu girdileri dinamik olarak listelemek. Ayrıca bu tutulan girdileri yani görevleri tamamlanma veya tamamlanma'ma durumlarını göre ekrana filtreleme yapılması.
